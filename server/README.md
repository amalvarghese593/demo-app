### .env

```bash

PORT=specifies the port of the backend server.
Example:
PORT=3010

MONGODB_URI=specifies mongo connection uri of the mongoDB database
Example:
MONGODB_URI=mongodb://localhost:27017
MONGODB_URI=mongodb+srv://<username>:<password>@<host>

MONGODB_NAME=specifies database name that backend server needs to connect to
Example:
MONGODB_NAME=demo-app

JWT_SECRET=specifies jwt secret used to sign the jwt token
Example:
JWT_SECRET=chsgchgqye8d9wyhuo

JWT_EXPIRES=specifies expiration time for the jwt token
Example:
JWT_EXPIRES=1d


```
