### .env

```bash

REACT_APP_BASE_URL=specifies url of backend server with ip and port
Example:
REACT_APP_BASE_URL=http://localhost:3010/api/v1

REACT_APP_REDUX_DEBUG=specifies if redux devtools need to be enabled or not for debugging purpose. If set to true, redux devtools would be enabled. This is an optional env.
Example:
REACT_APP_REDUX_DEBUG=true

```
